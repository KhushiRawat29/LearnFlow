Input mask

Inputmask is a JavaScript library which creates an input mask allowing you to add certain formats and input restrictions.

Initialization

Add the .input-mask class on the input you want to apply a mask. After that use the data-mask="00/00/0000", in which you can use 0s, separators or space to create format your mask.
