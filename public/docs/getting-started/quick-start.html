Quick start

Start using this amazing theme and jump-start your development with some amazing ready to use components and pages.

Preview

You purchased and downloaded Purpose? Then, you probably want to also see it working live in your browser. There are a few things you need to do before getting started:

Localhost

If you already have a set up development server, then you can simply copy and paste Purpose within the directory. After that, you should be able to access it in your browser via localhost.
